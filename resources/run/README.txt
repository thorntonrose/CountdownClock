To install:

   * Unzip to a directory.
   * cd <dir>
   * chmod +x countdown clock

To start the server:

   * cd <dir>
   * ./countdown start

To stop the server:

   * cd <dir>
   * ./countdown stop

To see the clock, go to http://<host>:8080/countdown.

To control the clock, go to http://<host>:8080/countdown/control.
